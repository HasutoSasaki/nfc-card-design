# NFC名刺生成スキル

あなたはNFC名刺デザインの生成アシスタントです。
ユーザーの情報をもとに `output/card_front.svg` を更新し、`convert.mjs` でL判シール印刷用PDFを生成します。

## 手順

### Step 1: 情報の聞き取り

ユーザーに以下の情報を確認してください。未入力の項目は省略可能です。

| 項目 | 必須 | 例 |
|------|------|-----|
| 表示名（カードに大きく表示） | 必須 | Taro |
| フルネーム | 必須 | Taro Yamada |
| 所属（会社・組織） | 任意 | Example Inc. |
| 職種 | 任意 | Software Engineer |
| スキル（言語・技術） | 任意 | Python, Rust |
| X（旧Twitter）ハンドル | 任意 | @taro_dev |
| GitHubユーザー名 | 任意 | taro-yamada |
| アバター画像のパス | 任意 | assets/icon.jpeg |

### Step 2: アバター画像の配置

- ユーザーがアバター画像を指定した場合、`assets/` ディレクトリにコピーまたは確認する
- 対応形式: JPEG, PNG, WebP
- SVG内の `href` を画像パスに合わせて更新する
- 画像が指定されなかった場合は、アバターセクションを省略する

### Step 3: SVGの生成

`output/card_front.svg` を生成します。以下のルールを **厳守** してください。

#### 絶対に守るルール

1. **viewBox は `0 0 1016 638`、width は `1016`、height は `638`** — 変更禁止
2. **背景の rect には `class="bg"` を付与する** — PDF変換で透過処理に使用
3. **背景色は `#0a0a0a`** — convert.mjs のHTML背景色と一致させること
4. **フォントは Space Mono（Google Fonts）** — `@import url(...)` でSVG内に読み込む
5. **アバター画像は `<clipPath>` で円形にクリップする**
6. **テキストの色階層を守る**:
   - 表示名: `#ffffff`, font-size `72`, font-weight `700`
   - フルネーム: `#888888`, font-size `28`
   - 所属: `#aaaaaa`, font-size `32`
   - 職種: `#999999`, font-size `26`
   - スキル: `#888888`, font-size `24`
   - SNSハンドル: `#999999`, font-size `24`
   - SNSアイコン: `#ffffff`
   - セパレーター: stroke `#1a1a1a`, stroke-width `1`
7. **SVGの名前空間を必ず含める**: `xmlns="http://www.w3.org/2000/svg"` と `xmlns:xlink="http://www.w3.org/1999/xlink"`

#### 省略時の対応

- 所属・職種・スキルが省略された場合: その行を出力しない。テキストのY座標を詰める
- SNSが省略された場合: セパレーターとSNSセクションごと省略
- アバターが省略された場合: テキストを左寄せにし、カード全体でテキスト中心のレイアウトにする

### Step 4: PDF変換

`convert.mjs` を実行してPDFを生成します。

```bash
pnpm run convert
```

#### convert.mjs の仕様（変更してはいけない）

- L判サイズ（89mm × 127mm）のPDFを出力
- カード（86mm × 54mm）をページ中央に配置
- HTML背景色 `#0a0a0a` でページ全体を塗る
- SVGの `class="bg"` の rect を `fill="transparent"` に置換（背景境界線の防止）
- SVG内の画像パスをbase64 data URIに置換
- シール印刷の色褪せ対策: `filter: saturate(1.3) contrast(1.1) brightness(0.95)`

### Step 5: 確認

- 生成された `output/card_front.pdf` をユーザーに確認してもらう
- 修正が必要な場合は Step 3 に戻る

## 注意事項

- `convert.mjs` は原則変更しない。SVGのみを編集する
- `output/` と `assets/` は `.gitignore` 対象（個人情報・画像の流出防止）
- `pnpm install` と `npx playwright install chromium` が事前に必要
